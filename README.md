An intelligent deep learning–based system for automatic pothole detection using modern object detection models. This project enables detection of road damage from images and videos and provides a simple Streamlit web interface for real-time usage.

The following models were trained and compared:

YOLOv8 → Best performance (mAP@0.5 ≈ 0.81)
YOLOv10 → Competitive performance
RT-DETR → Transformer-based detection

Final Model Used: YOLOv8 (highest accuracy and stability)
